# Behave Test Automation Framework for Python

## INSTALL
pip3 install behave

## USAGE
- Define steps in Gherkin language.
- Implement steps.
- Use tags to group features and scenarios. 
- Report.

## COMMANDS
   `$ behave`
   `$ behave --tags=tag`
   `$ behave --tags=~excludeTag`
   `$ behave features/featureFile.feature`

## REPORT
   `$ behave --junit`
